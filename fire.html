2. ΑΝΑΜΜΑ ΦΩΤΙΑΣ

Πρίν αρχίσετε το άναμμα μιας φωτιάς, εξετάστε τα παρακάτω ερωτήματα:

  • Ο χρόνος που πρόκειται να μείνετε στην παρούσα θέση δικαιολογεί το άναμμα μιας φωτιάς;
  • Έχετε πράγματι ανάγκη μιας φωτιάς;
  • Υπάρχει αρκετή καύσιμη ύλη πλησίον για να συντηρήσετε τη φωτιά;
  • Μήπως βρίσκεστε σε μια περιοχή όπου η φωτιά θα μπορούσε εύκολα να εξαπλωθεί και να τεθεί εκτός ελέγχου;
Θέση Φωτιάς

Η θέση για το άναμμα φωτιάς, πρέπει να επιλέγεται προσεκτικά, ιδιαίτερα όταν σχεδιάζετε να κατασκευασετε ένα καταφύγιο ή αν πνέει ισχυρός άνεμος. Η θερμότητα πρέπει να παρέχει ζεστασιά για το καταφύγιο σας, αλλά κατά τέτοιο τρόπο ώστε να μη σας τυλίξει ο καπνός (αν και λίγος καπνός βοηθάει στην απώθηση των εντόμων που τσιμπάνε). Η κατασκευή ενός ανεμοθραύστη θα εμποδίζει τον αέρα να σβήσει τις πρώτες φλόγες. Επίσης, θα περιορίζει την κατανάλωση της καύσιμης ύλης και θα αντανακλά περισσότερη θερμότητα στο καταφύγιο σας.

Το έδαφος κάτω από την φωτιά πρέπει να είναι στεγνό και χωρίς βλάστηση, για να εμποδίζει την εξάπλωση της φωτιάς. Αν υπάρχουν πετρες, φτιάξτε ένα κύκλο γύρω από την φωτιά, μόλις αυτή ανάψει καλά. Το μέτρο αυτό μειώνει τον κίνδυνο μετάδοσης της φωτιάς στο καταφύγιο σας ή στο γύρω χώρο και περιορίζει επίσης, το μέγεθος της φωτιάς και την κατανάλωση καύσιμης ύλης.

Αν η φωτιά δεν συντηρείται κατά την διάρκεια της νύχτας ή όταν θα πηγαίνετε για κυνήγι, μπορεί πιθανότατα να σβήσει. Οι περισσότερες φωτιές μπορούν ανα ανάψουν ξανά. αν τοποθετήσετε προσανάμματα πάνω στη παλιά θράκα και τα φυσήξετε. Το έδαφος κάτω από μια φωτιά θα παραμείνει ζεστό για πολλές ώρες. Αυτό βοηθαέι να ανάψετε γρήγορα μια νέα φωτιά.

Η φωτιά παίζει επίσης έναν πολύ σημαντικό ψυχολογικό ρόλο στην επιβίωση. Η ικανότητα για άναμμα φωτιάς αποδεικνύει στον επιζήσαντα ότι μπορεί να ελέγξει, τουλάχιστον, μερικά στοιχεία της κατάστασης και του παρέχει την άνεση της ζεστασιάς και του φωτός.

ΤΥΠΟΙ ΦΩΤΙΑΣ

Μόλις ανάψετε μια φωτιά πρέπει να βεβαιωθείτε ότι είναι κατάλληλη για τις ανάγκες σας. Αν είστε μόνος θα χρειαστεί μια μικρή φωτιά για να σας προσφέρει ζεστασιά. Οι μικρές φωτιές χρειάζονται λιγότερη καύσιμη ύλη, για να συντηρούνται και ελέγχονται ευκολότερα.

Αν βρέχει ή χιονίζει - και έχετε τα κατάλληλα μέσα - πρέπει να εξετάσετε την δυνατότητα να μεταφέρετε την φωτιά μέσα στο καταφύγιο σας. Ο καλύτερος τρόπος για να το κάνετε αυτό είναι να φτιάξετε μια αυτοσχέδια σόμπα. Μπορείτε επίσης να ανάψετε μια φωτιά για το μαγείρεμα της τροφής σας, ενώ εσεις θα έχετε πάει για κυνήγι. Επίσης, θα ζεσταίνει το κρεβάτι σας σε μια κρύα νύχτα.

Χρησιμοποίηση τρυπανιού για άναμμα φωτιάς
  • Το ιδανικό ξύλο για την βάση της σανίδας είναι ένα παλαιό, αποξηραμένο και εύκολα τριβόμενο ξύλο.
  • Το ιδανικό τρυπάνι κατασκευάζεται από ένα γερό και στεγνό κομμάτι από σκληρό ξύλο.
  • Χρησιμοποιείστε το καλύτερο διαθέσιμο τεχνητό προσάναμα.
  • Πάρτε άνετη θέση έτσι ώστε να μπορέσετε να εργασθείτε με το τρυπάνι επί ώρες, να χρειαστεί.
  • Μην εγκαταλείπετε την προσπάθεια - θα χρειαστείτε ώρες υπομονής.
  • Να έχετε δίπλα σας επιπλέον τεχνητό προσάναμα, για να το προσθέσετε όταν δείτε καπνό.
  • Να έχετε επίσης κοντά σας λεπτά φυσικά προσανάμματα, για να τα προσθέσετε αμέσως όταν δημιουργηθεί φλόγα από τα τεχνητά προσανάμματα.
  • Προστατεύετε και τροφοδοτείτε τα καιόμενα φυσικά προσανάμματα μέχρις ότου μπορέσετε να προσθέσετε μικρά ξερά κομμάτια καύσιμης ύλης.
